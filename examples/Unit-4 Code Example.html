Unit-4 Functions with Parameters - Code Example

You have been using functions that require or expect a parameter. Another way to say is that you already know how to call functions and pass a value to that function.

For example:

alert("Hello Class!"); The string "Hello Class!" is the value you passed to the alert( ) function to display.

prompt("Please enter your email address:", "name@server.edu"); This function can accept one or two parameters depending upon how you wish to use it.

document.write("<h1>Hello Class</h1>"); The string "<h1>Hello Class</h1>" is a value being passed to the write( ) of the document object in order to place it on the document object.
